# DaloWeb — Webs, Apps y Software a Medida

Landing page de **DaloWeb**, agencia digital especializada en diseño web profesional, aplicaciones móviles, sistemas de reservas online y software a medida.

## Estructura del proyecto

```
daloweb/
├── index.html                  # Redirección a la landing page
├── public/
│   ├── landing/
│   │   ├── index.html          # Landing page principal
│   │   ├── css/
│   │   │   └── styles.css      # Estilos de la landing
│   │   ├── js/
│   │   │   └── main.js         # JavaScript (animaciones, interactividad)
│   │   └── img/
│   │       └── projects/       # Capturas de proyectos destacados
│   └── src/
│       └── img/
│           └── favicon/        # Favicon del sitio
└── README.md
```

## Secciones

- **Hero** — Presentación con efecto typewriter y mockups animados.
- **Servicios** — Webs profesionales, apps móviles, sistemas de reservas, aplicaciones a medida.
- **Cómo trabajamos** — Proceso en 4 pasos: análisis, diseño funcional, personalización, publicación.
- **Estadísticas** — Proyectos entregados, valoración de clientes, años de experiencia.
- **Proyectos destacados** — Casos de éxito con capturas y resultados.
- **Tecnologías** — Marquee animado con el stack tecnológico.
- **Por qué DaloWeb** — Diferenciadores de la agencia.
- **Precios** — Planes transparentes para cada tipo de proyecto.
- **Contacto** — Formulario de contacto y presupuesto.

## Tecnologías

- HTML5
- CSS3 (variables CSS, flexbox, grid, animaciones)
- JavaScript vanilla (sin frameworks)
